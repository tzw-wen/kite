The swallow.exe was an application working on linux platforms to process .sqt files generated by sequest and prolucid.

The sandmartin.exe was an application working on linux platforms to combine results from swallow.exe

This application was part of the kite pipeline, which could schedule MS/MS searches or transfer the results into sqt file for swallow.exe.

The swallow.exe and sandmartin.exe used DTASelect/Contrast from John Yates group to make contrasts from experiments.  

The DTAelect/Contrast could compile hundreds of experiments into one projects. Please install DTASelect and put DTASelect and Contrast in your PATH.

Please use bash for all linux command in kite pipeline.

The swallow.exe could also use TPP and percolator for fdr control. Please install their commands into the Linux64 folder.

Please read the README-TPP.txt and percolator-license.txt for the license.

Please read the README in libsvm for libsvm.

Please use the command ldd SQTMassFixer to check the common libraries are available in your linux system.

The boost libraries were included with its license.
